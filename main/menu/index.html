Description

Created a functional website that allows users to track and plan their meals
Essentially how it works is that the server runs on an EC2 instance using linux to run the node server
However because the linux server is so old I had to essentially learn docker in order for the correct dependencies to run
That was a fun experience yet it taught me how AMAZING it is and how easy Docker makes my life and transporting files
Anyhow the script consists of Scraping the Michigan Dining Hall pages using Puppeteer.js then I Port it to Port 3000 using Express.js and CORS
however the most complicated problem was that my EC2 Server's public url was under HTTP so I had to learn how to move it to HTTPS
using NGINX and purchasing a domain and what not but it ended up well.

The frontend is raw HTML and CSS and primarily consists of multiple selections and features to select menus from different dining halls,
Curate a custom generated menu based on nutrition parameters, and track your macros.
