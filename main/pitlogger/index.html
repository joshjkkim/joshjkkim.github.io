Description

Hypixel Pit is a gamemode on the largest Minecraft server: Hypixel
To summarize the gamemode here is a short video:



This gamemode is the closest thing to pure anarchy and fighting within
this server and has grown quite the cult following
Therefore it was inevitable that many enemies and alliances were to be formed.

So I created this script that utilizes mineflayer.js and
discord.js to control several minecraft bot accounts
That seek and find individual pit lobbies and log all active players
Beyond the complex sorting system of which bots find individual
servers, there is a reminding system
So that users will get pinged if certain users get online and
can use commands to be notified for specific players
This is hosted on an EC2 Instance using Docker to host the server.
